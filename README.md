# TodayHeadlines
重庆邮电大学红岩网校2018寒假考核---仿今日头条\
## 知识点
* 用fragment填充ViewPager，对于需要频繁更新数据或网络请求的，适配器应继承FragmentStatePageradapter\
静态的页面继承Fragmentageradapter
* 封装使用HttpURLConnection实现网络请求，定义回调接口处理返回的信息
* json解析
